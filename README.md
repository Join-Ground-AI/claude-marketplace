# Ground Acquisition Skills for Ecommerce

AI-powered acquisition workflows for DTC and ecommerce teams — UGC campaigns, SEO content, and affiliate program management.

## Available Plugins

### Ground Acquisition Skills
13 AI workflow systems covering UGC campaigns, SEO content, and affiliate program management.

## Install

**Step 1: Add the marketplace**
```bash
claude plugins marketplace add Join-Ground-AI/Ground-Acquisition-Skills
```

**Step 2: Install the plugin**
```bash
claude plugins install ground-acquisition-skills@ground-acquisition
```
